# Website UAS Pemrograman Web 1

Website ini dibuat untuk memenuhi tugas UAS mata kuliah Pemrograman Web 1.
Website ini terdiri dari 5 halaman utama dan menggunakan HTML, CSS, serta PHP sederhana.

## ✅ Halaman Utama:
- `index.html`      → Home (Beranda)
- `gallery.html`    → Gallery (Galeri Gambar)
- `profile.html`    → Profile (Profil Mahasiswa)
- `form.php`        → Formulir Input (menggunakan PHP)
- `output.php`      → Output hasil form
- `contact.html`    → Contact Us (Informasi Kontak)

## 🎨 Fitur:
- Navigasi antar halaman (navbar)
- Input data menggunakan Form PHP
- Tabel data mahasiswa (di halaman profile)
- Gambar di halaman gallery

## 📁 Struktur Folder:
```
uas-web1/
├── index.html
├── gallery.html
├── profile.html
├── contact.html
├── form.php
├── output.php
├── css/
│   └── style.css
├── img/
    └── pic1.jpg, pic2.jpg (tambahkan sendiri)
```

## ▶️ Cara Menjalankan:
1. Ekstrak folder ini ke dalam `htdocs` (XAMPP)
2. Jalankan Apache via XAMPP
3. Akses melalui browser ke `http://localhost/uas-web1/`
